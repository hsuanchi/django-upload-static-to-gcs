# django-upload-static-to-gcs

## Launch on local runtime
Create your virtual environment
```
$ python3 -m venv venv
```

And enable virtual environment
```
$ . venv/bin/activate
```

Install requirements
```
$ pip install -r requirements.txt 
```

Run Django
```
$ python3 manager.py runserver```
```
